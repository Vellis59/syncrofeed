# Syncrofeed — MVP Promise

## What it is
Syncrofeed is a self-hosted RSS reader with a modern UI, smart reading features, and compatibility with existing mobile apps via the Fever API.

## Target user
Solo makers, developers, and tech-savvy people who self-host their tools and want control over their reading experience.

## Core product objects (MVP)
1. **Feeds** — RSS/Atom/JSON Feed subscriptions
2. **Articles** — individual entries from feeds (read/unread/starred)
3. **Collections** — user-defined groups of feeds (folders/tags)
4. **Digest** — a daily summary view of top articles

## MVP scope
- Add/remove feeds (URL or OPML import)
- Read articles in a clean reader view
- Mark as read/unread, star/favorite
- Organize feeds into collections
- Fever API endpoint for mobile app sync
- Dark/light theme
- Responsive + PWA

## Post-MVP (deferred)
- AI summaries
- Full-text search
- Multi-user support
- Google Reader API
- Sharing/bookmarking integrations
- Keyboard shortcuts power-user mode
